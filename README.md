# 🎨 Color Changer Project

A simple and interactive Color Changer web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Change background color instantly
- Simple and responsive UI
- Built with pure HTML, CSS, and JavaScript
- Beginner-friendly project

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📂 Project Structure

├── index.html
├── style.css
├── script.js

## 🎯 How It Works

1. Open the application in your browser.
2. Click the color change button.
3. The background color changes automatically.
